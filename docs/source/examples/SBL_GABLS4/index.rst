Stable BL: GABLS4
=================

The GABLS4 case simulates an extremely stable boundary layer at Dome C,
Antarctica (75.1°S, 123.3°E, elevation 3233 m) using Stage 3 initialization
data from the GABLS4 intercomparison project. The 24-hour simulation is driven
by time- and height-varying geostrophic winds and a prescribed surface
temperature derived from observed skin temperature, under a dry, advection-free
configuration. The reference domain is 1000 m × 1000 m × 1000 m.
